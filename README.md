# Flow

Flow is a growing icon set designed for interfaces. Every icon ships in two states — outline and fill — so you can match the visual weight of your UI.

**Format:** SVG, 24×24 viewBox
**States:** `icon-name.svg` (outline) · `icon-name-fill.svg` (fill)

---

## Usage

```html
<!-- Outline -->
<img src="icons/heart.svg" alt="Heart">

<!-- Fill -->
<img src="icons/heart-fill.svg" alt="Heart filled">
```

Or inline as SVG to control color via `currentColor`:

```html
<svg width="24" height="24" viewBox="0 0 24 24" fill="none">
  <!-- paste icon path here -->
</svg>
```
